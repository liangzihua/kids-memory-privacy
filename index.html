小记忆

儿童学习记忆辅助应用
隐私政策 · 最后更新：2025年1月1日
✅ 小记忆不收集任何个人信息，所有数据仅存储在您的设备本地。

一、我们收集哪些信息

小记忆应用不收集、不存储、不上传任何用户个人信息。所有学习数据（学习档案、卡片内容、学习进度、设置）均存储在用户设备的本地数据库中，不会传输到任何服务器。

二、网络权限用途

应用仅在以下用户主动触发的情况下使用网络：

  • 下载词库：用户在词库商店主动点击下载时，从公开数据源获取词库文件
  • AI 智能生题：用户配置 API Key 并主动点击生成时，调用第三方 AI 接口
  • 语音识别：调用设备系统内置的语音识别服务

应用不会在后台进行任何网络请求，不存在数据上报或行为追踪。

三、麦克风权限用途

仅用于语音跟读练习、语音背诵评分、语音切换档案功能。所有录音数据不保存、不上传，处理完成后立即丢弃。

四、相机权限用途

仅用于拍照导入题目（OCR 解析）。照片不保存、不上传，解析完成即丢弃。

五、儿童隐私保护

本应用专为儿童设计，严格遵守《儿童个人信息网络保护规定》：

  • 不收集任何儿童个人信息（姓名、年龄、位置等）
  • 不要求用户注册账号
  • 不进行行为画像或精准推送
  • 不包含任何第三方广告 SDK

六、数据安全

所有数据存储在用户设备本地。用户可在设置中随时清除全部数据；卸载应用后系统自动清除所有本地数据。我们不设服务器、不建用户账号、不追踪用户行为。

七、第三方服务（可选）

  • AI 接口（DeepSeek/通义千问/豆包等）：用于 AI 生题，需用户自行填写 API Key
  • Azure 认知服务：可选用于发音评分，需用户自行填写 Key
  • 系统语音服务：TTS 朗读和 STT 识别使用设备系统内置引擎

八、联系我们

如对本隐私政策有任何疑问，请通过华为应用市场的开发者联系方式与我们沟通。
